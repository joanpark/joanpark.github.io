# 멀티플레이 게임 개발 in 유니티 & 미러(Mirror) (1)

## 미러(Mirror) 에 대한 소개

유니티용으로 제작된 하이 레벨 Network API
하위 레벨에서 Telepathy라는 C# TCP Library를 이용하고 있다.
미러와 텔레파시 모두 vis2k라는 팀에서 제작하였는데, 심플하면서도 강력한 네트웍 기능을 제공한다.
